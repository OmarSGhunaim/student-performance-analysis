# Student Performance Analysis

## Description
This project analyzes student academic data to identify key factors that influence performance. Using data analysis and visualization techniques, the project explores relationships such as study time and final grades.

## Objectives
- Understand patterns in student performance
- Identify factors that impact academic success
- Visualize data for better insights

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Key Features
- Data cleaning and exploration
- Statistical analysis
- Visualizations (bar plots, box plots)

## Results
The analysis shows that increased study time generally leads to higher grades, demonstrating a clear relationship between effort and performance.

## Conclusion
This project demonstrates fundamental data analysis skills, including working with datasets, visualizing trends, and drawing meaningful conclusions.
